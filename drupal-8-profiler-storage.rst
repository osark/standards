Drupal 8 Profiler Storage
=========================

Requirements
------------

-  Drupal 8
-  Webprofiler (*Part of Devel Module*)

Issue
-----

Error message: "*Unable to store profiler information*\ "

Possible Solutions
------------------

1. Storage Backend

-  Go to ``/admin/config/development/devel/webprofiler``
-  Change *Storage backend* to **File storage**.
